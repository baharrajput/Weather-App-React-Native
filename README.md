# Weather Forecast Mobile App - React Native

## Overview

This project is a modern Weather Forecast mobile application developed using React Native. The application displays weather information with hourly forecasts, weekly forecasts, and location search functionality using a clean and responsive mobile interface.

## Features

* Hourly Weather Forecast
* Weekly Weather Forecast
* Location Search Screen
* Modern UI Design
* Bottom Tab Navigation
* Responsive Mobile Interface
* Splash Screen Navigation

## Technologies Used

* React Native
* TypeScript
* React Navigation
* React Native Vector Icons

## Screens Included

* Splash Screen
* Home Screen
* Weekly Forecast Screen
* Hourly Forecast Screen
* Location Search Screen

## Project Structure

* Components
* Navigation Screens
* Assets & Icons
* Forecast Modules
